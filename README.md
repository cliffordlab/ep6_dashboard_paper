# Dashboard for Executive Park 6

## Requirements 

Install the necessary python packages for the backend

```bash
cd backend
pip install -r requirements.txt

```

Install the NPM packages necessary

```bash
cd ../dashboard
npm install .
```

## Run the Servers


Run the Flask server at 

```bash
cd backend
python run.py
```

Run the Dashboard at 

```bash
cd dashboard
npm start
```
